LINKING JAVASCRIPT WITH HTML
1. Embedding the javascript code inside the Html document, which is called inline javascript.
    <script>
    //Script logic
    </script>
2. Linking external javascript files using src attribute of the <script> tag, this way of linking javascript is called external javascript.
    <script src="myscripts.js"></script>

When to Use var, let, or const?

1. Always declare variables
2. Always use const if the value should not be changed
3. Always use const if the type should not be changed (Arrays and Objects)
4. Only use let if you can't use const
5. Only use var if you MUST support old browsers.
