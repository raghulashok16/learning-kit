1. JSX
    -- javascript XML
    -- allows writing HTML structures in javascript file using XML-like syntax.
2. components
    -- independent, reusable pieces of UI.
    -- can be functional or class - based.
3. props
    -- short form of properties.
    -- they are read-only inputs to the components that defines attributes or configuration.
4. state
    -- holds data that might change over the lifecycle of a component.
    -- used in clas components and functional components via the useState hook.
5. higher-order-components(HOC)
    -- a function that takes a component and returns a new component, used for reusing component logic.
