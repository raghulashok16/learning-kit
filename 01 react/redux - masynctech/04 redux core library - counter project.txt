# ---------------------------- Rules of a Reducer ---------------------------- #
âœ… 1. Reducer - it is a function that recevies the current state and an action object, decides how to update the state base on the action, and returns the new state.
âœ… 2. the current state is the state inside the STORE.

RULES:
âœ… 1. the new state value should only be calculated based on the state and action arguments.
âœ… 2. reducers are not allowed to modify the existing state.
