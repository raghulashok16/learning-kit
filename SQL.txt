WHAT IS A DATABASE ?

<------------------------------->

WHAT IS SQL ?

~> SQL is a language that is used to talk to a databases.
~> SQL statement is nothing but a Query
~> SELECT * FROM USERS
~> the above is the Query or SQL Statement
<------------------------------->

QUERY BREAKDOWN:

SELECT NAME FROM USERS WHERE ROLE = 'MANAGER';

NAME --> identifier
WHERE --> keyword
ROLE = 'MANAGER' --> condition
'MANAGER' --> expression
